# Sistema Gestione Presenze - Fenice

Sistema completo per la gestione delle presenze dei dipendenti con app mobile Android, pannello admin web e API REST.

## 🎯 Caratteristiche Principali

### 📱 App Mobile (Flutter)
- ✅ Login offline con PIN alfanumerico
- ✅ Timbratura con geolocalizzazione GPS
- ✅ Geocoding inverso (coordinate → indirizzo)
- ✅ Logica intelligente pulsanti (ENTRATA → PAUSA → USCITA)
- ✅ Supporto turni multipli nella stessa giornata
- ✅ Richiesta ferie/permessi dall'app
- ✅ Visualizzazione saldo ferie con maturazione mensile
- ✅ Notifiche push per approvazioni/rifiuti
- ✅ Mappa delle timbrature con OpenStreetMap
- ✅ Export timbrature in Excel
- ✅ Sincronizzazione bidirezionale (online/offline)

### 🌐 Pannello Admin Web (PHP)
- ✅ Login sicuro con password hash bcrypt
- ✅ Dashboard con statistiche e filtri
- ✅ Gestione dipendenti (aggiungi/modifica/elimina)
- ✅ Workflow ferie (richiesta → approvazione/rifiuto)
- ✅ Calendario visuale con colori per dipendente
- ✅ Mappa admin con tutte le timbrature geolocalizzate
- ✅ Statistiche dettagliate per dipendente e azienda
- ✅ Configurazione maturazione ferie/permessi personalizzata
- ✅ Sistema notifiche completo
- ✅ Export CSV

### 🔌 API REST (Node.js)
- ✅ Autenticazione con token
- ✅ Gestione timbrature
- ✅ Workflow ferie con notifiche automatiche
- ✅ Geocoding inverso con Nominatim (OpenStreetMap)
- ✅ Calcolo dinamico maturazione ferie
- ✅ Sistema notifiche
- ✅ Health check per verifica connessione

### 🗄️ Database (MySQL)
- ✅ Struttura relazionale ottimizzata
- ✅ Supporto workflow ferie con stati
- ✅ Configurazione personalizzata per dipendente
- ✅ Tracciamento notifiche
